# SweetSlayer
#是什么
[Emlog5.3.1](http://www.emlog.net) 一款响应式主题模板。

#为什么
我在我的[一篇博客](http://pinruan.net/?post=33)中介绍过这款主题的产生原因。

#特点
1,跨平台自动适配
2,采用 foundation5 作为框架
3,支持代码高亮
4,删去一切不必要的元素，比如 archieves,calendar,最新文章 这些看似有用的东西
5,个别元素可自行配置 参看 ssconfig.php 注释。
